## What you need:
1) Nerd Fonts
2) Make a apiKey(There is https://www.weatherapi.com)
3) Create a config file with path like that: /home/yourname/.config/wfic/wfic.conf

## How to use:
sh .../wfic.sh "CityName" "DaysNumber"

## Config example
textColor=5;
daysColor=10;
temColor=11;
nameColor=10;
interval=4;

nightColor=1;
eveningColor=6;
morningColor=14;

userName=Timur;
apiKey=xxxApiKey;
